**Direction:** down

|bps|直播视频码率|int|||
|dpi|直播视频分辨率|text|||

```json
{
	"method": "live_set_detail",

	"data": {
		"video_id": "AFAFEFA34D/0-43-0/normal-1",
		"fps": 30,
		"bps": 3000000,
		"dpi": "1920x1080"
	}
}
```
