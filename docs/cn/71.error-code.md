
## 错误码说明
1. 整体而言，错误码为 “ABCDEF” 格式，A 说明上报来源，BC 说明功能模块，DEF 为模块自定义错误。
2. HMS 的错误描述不能直接通过错误码得到。HMS 的异常描述的获取，需要通过 HMS 错误码按照 [HMS 功能](https://developer.dji.com/doc/cloud-api-tutorial/cn/feature-set/dock-feature-set/hms.html)章节介绍的规则完成`文案Key`的拼接，依靠`文案Key`在[hms.json](https://terra-1-g.djicdn.com/84f990b0bbd145e6a3930de0c55d3b2b/apicloud/hms.json)中查询告警文案，告警文案中有变量的需要按照规则替换。
