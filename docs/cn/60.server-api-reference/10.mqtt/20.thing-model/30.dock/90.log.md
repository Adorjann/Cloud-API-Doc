### 发起日志文件上传
**Topic:** thing/product/{pid}/services

**Direction:** down

**Method:** fileupload_start

**Data:**
|Column|Name|Type|constraint|Description|

|credentials|凭证信息|struct||| 
|»access_key_id|访问密钥ID|text|{}||
