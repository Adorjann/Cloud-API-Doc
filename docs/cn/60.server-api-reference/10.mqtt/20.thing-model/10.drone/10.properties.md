## Properties

* `push_mode = 0`：设备端主动定频推送信息
* `push_mode = 1`：设备端不做定频推送，但是如果该Property发生变化，需要立即推送服务端

| --------------------------------------- | -------------- | ------------------------------------------------------------ | --------- | --------- |

| `gear`                                  | 档位           | 见下表 档位说明                                              | 0         | int       |
| `battery`                               | 飞机电池       | 详情见电池状态说明                                           | 0         | struct    |

| `attitude_pitch`                        | 俯仰轴角度     | -                                                            | 0         | double    |
