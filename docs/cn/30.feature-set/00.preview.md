Pilot与机场都可以通过上云API协议实现功能，与第三方云平台进行数据交互。但两者在支持的功能类型、使用的协议等方面存在差异。为保证用户对上云API业务有更全面、更具体的理解，我们预先列举了上云API工作流程、支持的功能、对应功能用到的协议。同时用户可在后续章节查看更为详细的功能介绍。
