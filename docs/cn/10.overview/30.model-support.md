## 机型支持

| 飞行平台 | 网关设备                 | 负载                | 备注               |
| -------- | ------------------------ | ------------------- | ------------------ |
| M300 RTK | DJI RC + DJI Pilot 2      | H20T/H20 <br> P1/L1 | 三方负载暂时不支持 |
| M30T/M30 | DJI RC Plus + DJI Pilot 2 | --                  | 三方负载暂时不支持 |
| M30T/M30 | 大疆机场                 | --                  | 三方负载暂时不支持 |

> **注意：**
> 
> 上云API只支持M30系列、M300 RTK，暂不支持M200V2 系列、M200系列、M2E、P4R、M2EA等老机型。
